# Medicine Analytics and Price Prediction using Python

This project focuses on Exploratory Data Analysis (EDA), data preprocessing, feature engineering, and machine learning on a pharmaceutical medicine dataset. The analysis was performed using Python in Google Colab to understand medicine categories, dosage forms, manufacturers, package information, and pricing patterns.

The project includes:

* Data cleaning and handling missing values
* Duplicate checking and feature inspection
* Visualization of medicine types, dosage forms, and manufacturers
* Feature extraction from package size and strength columns
* Data transformation and preprocessing
* Machine learning model building using Random Forest Regressor
* Feature importance analysis for medicine price prediction

Tools & Libraries Used:

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

This project demonstrates practical skills in healthcare/pharmaceutical data analytics, exploratory data analysis, feature engineering, and predictive modeling workflows.
